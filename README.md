# Karabiner-Elements Keybinds

A place to track, backup, share the config that I use when docking my Mac to a Windows keyboard. 

Give MacOS that Windows feel but without any of the AI nonsense. 

Keyboard shortcuts should be able to be OS agnostic. 

Perhaps in the future an AutoHotkey script could be added to make windows behave like MacOs when it has a Mac keyboard plugged in, but I designate that permenatnly out of scope.
